_.toSentence (array, delimiter, lastDelimiter)
Converts an array of strings into a human-readable string

Parameters

  • array: an array
  • delimiter: a character used for separating each strings in the given array when concatenating them together. Defaults to ',' if not given.
  • lastDelimiter: another character used for separating the last string in the given array from the previous ones. Defaults to ' and ' if not given.

Usage:

 local arrayOfWords = {'banana','apple','mango','tomato','lemon'} 
 _.toSentence(arrayOfWords) --> 'Banana, apple, mango, tomato and lemon.' 
 _.toSentence(arrayOfWords,'.') --> 'Banana. apple. mango. tomato and lemon.' 
 _.toSentence(arrayOfWords,'.','. ') --> 'Banana. apple. mango. tomato. lemon.'

Return value:

a string
